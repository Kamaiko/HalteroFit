## Overall Interface Philosophy

**Design Approach:**

- Feature-rich and comprehensive (1,400+ exercises)
- Prioritizes functionality over sleek minimalism
- Community-focused with social features
- Web + mobile interface available

**Interface Characteristics (from reviews):**

- Described as "powerful but dated" compared to newer apps
- Can feel "crowded" or "overwhelming" for new users
- Learning curve exists due to extensive features
- More functional than visually sleek

**Target User:** Intermediate to advanced lifters who want comprehensive tracking, analytics, and community features

---

## Main Navigation Structure

### Bottom Tab Bar (Primary Navigation)

Based on research, Jefit uses a bottom navigation with the following main tabs:

1. **Workout** - Primary workout management
2. **Exercises** - Exercise library and database
3. **Progress** - Analytics and tracking
4. **Community** - Social features
5. **Profile** - User settings and preferences

## Screen Details

### 1. Workout Tab

The Workout tab has been recently revamped (December 2023) for clarity and better planning.

#### Sub-navigation (Top tabs)

**Find Workouts**

- Purpose: Discover pre-made workout programs
- Access: Bottom nav "Workout" → Top tab "Find"
- Features:
  - Library of pre-made workout programs
  - Filter by: Goal, Days, Fitness level
  - Tap program to view details and save

**My Plans**

- Purpose: Manage personal workout plans
- Access: Bottom nav "Workout" → Top tab "My Plans"
- Features:
  - View all saved workout plans
  - Shows current active routine
  - Quick access button: "Create My Own Workout Plan" (top of screen)
  - "Start Workout" button (primary CTA from My Plans page)
  - Entry-level routines provided by default

#### My Plans - Overview Screen

**Layout:**

- List of workout plans
- Current routine highlighted
- Day cards showing daily exercises
- Swipe/slide gesture to view daily exercise lists

**Actions:**

- Three-dot menu on each plan:
  - Edit (path: workout tab → my plans → overview → three dots → Edit)
  - Delete
  - Set as default routine

#### My Plans - Day Details

**Access:** Tap on a specific day from plan overview

**Layout:**

- List of exercises for that day
- Exercise order visible
- Set/rep information preview

**Actions:**

- Three-dot menu:
  - Edit (path: workout tab → my plans → day details → three dots → Edit)
  - Add exercises
  - Reorder exercises
  - Remove exercises

### 2. Exercise Library Screen

**Purpose:** Browse and search 1,400+ exercises

**Access:** Bottom nav "Exercises" tab

#### Search & Filter Interface

**Search Bar:**

- Magnifying glass icon or search bar at top
- Real-time search as you type

**Filter System:**

- Filter by Body Part/Muscle Group
  - Categories: Abs, Chest, Back, Legs, Shoulders, Arms, etc.
  - Visual muscle group icons
- Filter by Equipment
  - Categories: Body Only, Dumbbells, Barbells, Machines, Cables, etc.
- Filter by Exercise Type
  - Strength, Stretching, Cardio, etc.
- Multiple selection support
- "Apply" button to confirm filters

**Recent Enhancement (2024):**

- "Smarter exercise filters" rolled out to some users
- Improved filtering logic and UX

#### Exercise Card Layout

**Information Displayed:**

- Exercise name
- Main muscle group (body part)
- Equipment required
- Exercise type (Strength/Cardio/etc.)
- Video/GIF demonstration thumbnail

**Actions:**

- Tap exercise card to view:
  - Full demonstration video/animation
  - Written step-by-step instructions
  - Add to workout option

### 3. Active Workout Screen (Exercise Logging)

**Purpose:** Log sets in real-time during workout

**Access:** "Start Workout" button from My Plans

**Recent Redesign (December 2023):**

- Cleaner layout
- Updated timer interface
- Enhanced 1RM tracking

#### Screen Layout

**Top Section:**

- Exercise name and demonstration video/GIF
- Swipe up to fold video for focused logging
- Swipe horizontally to navigate between exercises
- Exercise progress indicator (e.g., "Exercise 2 of 5")

**Middle Section - Set Logging:**

- Table/list of sets
- Columns:
  - Set number (or "W" for warmup sets)
  - Previous performance (weight × reps from last workout)
  - Weight input field
  - Reps input field
  - Checkmark button to log set

**Data Pre-loading:**

- Auto-loads previous workout data
- Shows history for reference
- One-tap to copy previous set
- Individual rest time per set (customizable)

**Set Editing:**

- Can "jump and edit between sets without constraints"
- Pre-log set editing available
- Changes must be saved before leaving screen

**Bottom Section - Rest Timer:**

- Auto-starts after logging a set
- Swipe down to minimize (low profile mode)
- Timer remains accessible while minimized
- Enhanced interval timer accessibility
- Sound/vibration alerts

**Navigation Between Exercises:**

- Swipe left/right at top of screen
- Or use navigation arrows
- No need to exit screen to switch exercises

## Key Features by Screen

### Workout Planning Features

**Create Workout Plan:**

1. Navigate: Workout → My Plans → "All"
2. Tap "Create My Own Workout Plan" button (top of screen)
3. Name workout plan
4. Add days
5. Add exercises to each day
6. Set default sets/reps per exercise

**Download Pre-Made Plans:**

1. Navigate: Workout → Find
2. Browse or filter programs
3. Tap program to preview
4. Save to "My Plans"

### Set Logging Features

**During Workout:**

- Previous set data visible for reference
- One-tap set logging
- Auto-start rest timer after set
- Real-time 1RM calculation updates
- Warmup set marking ("W" indicator)
- Failure set marking
- Notes per set (optional)

**Timer Features:**

- Customizable rest duration per exercise/set
- Swipe to minimize/maximize
- Sound and vibration alerts
- Continues in background

### Exercise Discovery Features

**Search:**

- Text search by exercise name
- Real-time results filtering

**Browse:**

- By muscle group (visual icons)
- By equipment available
- By exercise type
- Multiple filters can be combined

**Exercise Details:**

- HD video demonstration
- Step-by-step written instructions
- Muscle activation visualization
- Equipment requirements
- Alternative exercises suggestions (inferred from reviews)

## UX Strengths and Weaknesses

### Strengths

**Feature Completeness:**

- 1,400+ exercise library (comprehensive)
- Extensive analytics and progress tracking
- Advanced 1RM calculations
- Community features for motivation
- Web interface for planning

**Functionality:**

- One-tap set logging once in workout
- Previous workout data pre-loaded
- Auto-start rest timer
- Offline functionality
- Cross-device sync

**Data Tracking:**

- Captures comprehensive workout data
- Advanced filtering and timeframe analysis
- Body measurement tracking
- Progress visualization

### Weaknesses

**Interface Complexity:**

- Can feel "crowded" or "overwhelming"
- Learning curve for new users
- Multiple screens to navigate for some tasks (improved in 2025)
- "Dated" visual design compared to competitors

**Navigation:**

- Pre-2025: Switching between screens to edit workouts was frustrating
- "Not intuitive" for adding and logging exercises initially
- Too many tabs and features can confuse navigation

**Visual Design:**

- Described as "more functional than sleek"
- Not as visually modern as Hevy or Strong
- Interface hasn't aged as well as minimalist competitors

### User Experience Notes

**From Reviews:**

- "Clean and intuitive once you learn it"
- "Powerful, but overwhelming at first"
- "The most features, but not the prettiest"
- "Navigation is a breeze once familiar with layout"

## Comparison with Competitors

### Strong

**Interface:** Minimalist, fast, clean, efficient

- Emphasizes speed and simplicity
- Less cluttered than JEFIT
- Mobile-only (no web interface)
- Streamlined workout logging

**Philosophy:** Do one thing (workout tracking) extremely well

### Hevy

**Interface:** Modern, sleek, intuitive

- Balance between features and design
- "Joy to use" according to reviews
- Most functions 2-3 taps away
- Web interface with folder organization

**Philosophy:** Modern design with comprehensive features

### JEFIT

**Interface:** Feature-rich, comprehensive, dated

- Extensive features can feel cluttered
- Web + mobile interfaces
- Community and social features
- More learning curve

**Philosophy:** All-in-one fitness platform (tracking + community + programs)
