## About

---

## Development

### Prerequisites

- Node.js 18+
- iOS Simulator or Android Emulator
- [Expo Development Build](https://docs.expo.dev/develop/development-builds/introduction/)

### Setup

```bash
npm install
cp .env.example .env.local  # Configure your environment
npm start
```
